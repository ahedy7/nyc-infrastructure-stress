# NYC Infrastructure Stress Analysis

This project analyzes infrastructure stress and capacity utilization across New York City.

## Overview

The NYC Infrastructure Stress project provides data analysis and visualization tools for understanding infrastructure load and resilience across the city.

## Technologies Used

- **Data Processing**: pandas, numpy, geopandas
- **Spatial Analysis**: shapely, pyarrow
- **Visualization**: plotly, streamlit, dash
- **Backend**: Python 3.11, gunicorn

## Installation

```bash
pip install -r requirements.txt
```
